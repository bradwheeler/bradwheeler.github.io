## Listening checklist

After each generation, listen for:

1. **Proper names** — the model frequently mispronounces unfamiliar names. Flag any name that sounds wrong and add a text substitution to `PRONUNCIATION_SUBS`.
2. **Voicing discontinuities** — unnatural pauses, stutters, or rhythm breaks mid-sentence. These are usually caused by punctuation, long compound phrases, or rare multi-syllable words. Fix by rewording the sentence or inserting a `<break time="Xs" />` tag.
3. **Acronyms** — the model may spell them out or mispronounce them. Add phonetic substitutions to `PRONUNCIATION_SUBS` as needed.

## Known pronunciation substitutions

| Original | Substitution | Notes |
|----------|-------------|-------|
| `Susser` | `Süsser` | Umlaut ü forces correct "SUH-ser"; plain "Su" always maps to "Soo" |
| `Arant` | `Arrant` | Correct phonetic match |
| `HMAP` | `eighchmap` | Reads the H as a letter, then "map" |

## Adding a `<break>` tag

The `<speak>` wrapper is added automatically by `prepare_text()`. Insert break tags directly in chapter text strings:

```python
"...end of thought.<break time=\"1.5s\" />Next thought..."
```

Max duration per tag is 3 seconds. Chain tags for longer pauses.

## Adding new chapters

Add an entry to the `CHAPTERS` dict in `narrate_chapters.py`:

```python
"chapter_key": {
    "title": "Chapter Title",
    "text": """Chapter text here."""
}
```

Output MP3 is saved to `audio/chapter_key.mp3`.
